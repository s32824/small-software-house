# Small Software House - Project Team Manager

A Java console application demonstrating core Object-Oriented Programming (OOP) concepts.

## Features
* **Inheritance:** Uses a base `Employee` class with specific `Developer` and `Tester` child classes.
* **Polymorphism:** Stores different employee types in a single `ArrayList` and calls overridden methods.
* **Encapsulation:** Uses private fields with appropriate getters and constructors.
* **Data Validation:** Overrides `equals()` to compare employees by their unique IDs.

## How to Run
1. Open the project in IntelliJ IDEA.
2. Locate the `Main.java` file.
3. Right-click and select **Run 'Main.main()'**.

- [x] Inheritance and `super`
